jQuery.type()

Categories: Utilities

jQuery.type( obj ) 返回： String

描述： 确定JavaScript 对象的类型。

  • version added: 1.4.3jQuery.type( obj )

    obj用来获取JavaScript数据类型的对象。

一些用来确定不同类型对象的数据类型。逻辑可以按以下方法确定：

  • 如果对象是undefined或null，则返回相应的“未定义”或“空”。
  • 如果对象有一个内部的[[Class]]相当于一个浏览器的内置对象我们返回关联的名称。 (有关此技术的更多细节。 )
    • jQuery.type(true) === "boolean"
    • jQuery.type(3) === "number"
    • jQuery.type("test") === "string"
    • jQuery.type(function(){}) === "function"
    • jQuery.type([]) === "array"
    • jQuery.type(new Date()) === "date"
    • jQuery.type(/test/) === "regexp"
  • 其他一切都将返回它的类型“object”。

Example:

Finds out if the parameter is a RegExp.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  Is it a RegExp? <b></b>
<script>$("b").append( "" + jQuery.type(/test/) );</script>

</body>
</html>

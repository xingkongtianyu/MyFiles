.resize()

Categories: Events > Browser Events

.resize( handler(eventObject) ) 返回： jQuery

描述： 为 "resize" 事件绑定一个处理函数，或者触发元素上的 "resize" 事件。

  • version added: 1.0.resize( handler(eventObject) )

    handler(eventObject)每次事件触发时会执行的函数。

  • version added: 1.4.3.resize( [ eventData ], handler(eventObject) )

    eventData将要传递给事件处理函数的数据映射。

    handler(eventObject)每次事件触发时会执行的函数。

  • version added: 1.0.resize()

这个函数的第一种用法是 .bind('resize', handler) 的快捷方式，第二种用法是 .trigger('resize') 的快捷方式。

当浏览器window的尺寸改变时，resize事件将被绑定到window元素：

$(window).resize(function() {
  $('#log').append('<div>Handler for .resize() called.</div>');
});

现在，只要在浏览器窗口的大小改变时，根据不同的浏览器，该消息被追加到<div id="log">一次或多次。

代码在resize处理函数不应该依靠多次处理程序被调用。根据执行情况，resize事件可以连续发送,作为正在进行中的重定尺寸(在Internet Explorer和Safari和Chrome等基于WebKit的浏览器的典型行为),或只在改变尺寸操作上只执行一次手术（在Firefox中的典型行为）。

Example:

To see the window width while (or after) it is resized, try:


$(window).resize(function() {
  $('body').prepend('<div>' + $(window).width() + '</div>');
});

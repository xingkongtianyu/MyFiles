deferred.resolveWith()

Categories: Deferred Object

deferred.resolveWith( context, args ) 返回： Deferred

描述： 解决递延对象，并根据给定的上下文和参数调用任何完成的回调函数。

  • version added: 1.5deferred.resolveWith( context, args )

    context 上下文作为this对象传递给 doneCallbacks 。

    args 传递给doneCallbacks的可选参数

当延迟解决，任何 doneCallbacks 添加的deferred.then或deferred.done被调用。回调按他们添加的顺序执行。每个回调传递的args在deferred.resolve()中调用。之后添加任何 doneCallbacks 递延被拒绝进入状态时，立即执行添加，使用的参数被传递给.resolve()调用。有关详细信息，请参阅文件Deferred object 。

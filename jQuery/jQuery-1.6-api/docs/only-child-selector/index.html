:only-child Selector

Categories: Selectors > Child Filter

only-child selector

version added: 1.1.4jQuery(':only-child')

描述： 选择所有其父元素下只有一个子元素的元素。

若家长有其他子元素，就不会被匹配。

Example:

Finds the button with no siblings in each matched div and modifies look.

<!DOCTYPE html>
<html>
<head>
  <style>

  div { width:100px; height:80px; margin:5px; float:left; background:#b9e }
  </style>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  <div>
    <button>Sibling!</button>
    <button>Sibling!</button>
  </div>

  <div>
    <button>Sibling!</button>
  </div>
  <div>
    None
  </div>

  <div>  
    <button>Sibling!</button>
    <button>Sibling!</button>
    <button>Sibling!</button>

  </div>
  <div>
    <button>Sibling!</button>
  </div>
<script>$("div button:only-child").text("Alone").css("border", "2px blue solid");</script>

</body>
</html>

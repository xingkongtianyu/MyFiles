event.relatedTarget

Categories: Events > Event Object

event.relatedTarget 返回： Element

描述： 在事件中涉及的其它任何DOM元素。

  • version added: 1.1.4event.relatedTarget

对于 mouseout 事件，它指向被进入的元素；对于 mousein 事件，它指向被离开的元素。

Example:

On mouseout of anchors, alert the element type being entered.

$("a").mouseout(function(event) {
  alert(event.relatedTarget.nodeName); // "DIV"
});

event.data

Categories: Events > Event Object

event.data 返回： Anything

描述： 当前执行的处理器被绑定的时候，包含可选的数据传递给jQuery.fn.bind。

  • version added: 1.1event.data

Example:

The description of the example.

$("a").each(function(i) {
  $(this).bind('click', {index:i}, function(e){
     alert('my index is ' + e.data.index);
  });
});

.hasClass()

Categories: Attributes | Manipulation > Class Attribute | CSS

.hasClass( className ) 返回: Boolean

描述: 检查匹配的元素是否含有某个特定的类。

  • version added: 1.2.hasClass( className )

    className要查询的样式名。

元素可能会有个的样式名。在HTML中这表现在多个样式名用空格隔开:

<div id="mydiv" class="foo bar"></div>

如果匹配元素上有指定的样式，那么.hasClass() 方法将返回 true ， 即使其他样式也是。 举个例子, 给上文的HTML加上下面的代码将返回 true：

$('#mydiv').hasClass('foo')

也可以:

$('#mydiv').hasClass('bar')

举例：

在匹配的元素上寻找 'selected' 样式。

<!DOCTYPE html>
<html>
<head>
  <style>

  p { margin: 8px; font-size:16px; }
  .selected { color:red; }
  .highlight { background:yellow; }
  </style>
  <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
</head>
<body>
	<p>Hello</p>
  <p class="selected">Goodbye</p>
  <div id="result1">First paragraph has selected class: </div>

  <div id="result2">Last paragraph has selected class: </div>
  <div id="result3">Some paragraph has selected class: </div>
<script>$("div#result1").append($("p:first").hasClass("selected").toString());
$("div#result2").append($("p:last").hasClass("selected").toString());
$("div#result3").append($("p").hasClass("selected").toString());</script>
</body>
</html>

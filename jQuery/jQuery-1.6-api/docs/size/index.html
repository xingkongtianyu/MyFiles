.size()

Categories: Miscellaneous > DOM Element Methods

.size() 返回： Number

描述： 返回的jQuery对象匹配的DOM元素的数量。

  • version added: 1.0.size()

假设页面上有下面一个简单的无序列表：

<ul>
  <li>foo</li>
  <li>bar</li>
</ul>

我们通过调用 .size() 来获取列表项的数目：

alert('Size: ' + $('li').size());

于是就显示了列表项的个数：

Size: 2

你也可以使用 .length 属性来代替，他会略微快那么一点点。

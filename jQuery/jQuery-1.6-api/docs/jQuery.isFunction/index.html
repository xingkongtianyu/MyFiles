jQuery.isFunction()

Categories: Utilities

jQuery.isFunction( obj ) 返回： boolean

描述： 确定的参数是一个Javascript 函数。

  • version added: 1.2jQuery.isFunction( obj )

    obj用于测试是否为函数的对象

注意：jQuery 1.3以后，在IE浏览器里，浏览器提供的函数比如alert()还有 DOM 元素的方法比如 getAttribute() 将不认为是函数。

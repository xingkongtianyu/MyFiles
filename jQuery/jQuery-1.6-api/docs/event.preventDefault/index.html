event.preventDefault()

Categories: Events > Event Object

event.preventDefault() 返回： undefined

描述： 阻止默认事件行为的触发。

  • version added: 1.0event.preventDefault()

例如，在执行这个方法后，如果点击一个锚点<a />，就不会让浏览器跳转到新的 URL 去了。我们可以用 event.isDefaultPrevented() 来确定这个方法是否(在那个事件对象上)调用过了。

Example:

取消点击的默认行为(导航)。

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  
<a href="http://jquery.com">default click action is prevented</a>
<div id="log"></div>

<script>
$("a").click(function(event) {
  event.preventDefault();
  $('<div/>')
    .append('default ' + event.type + ' prevented')
    .appendTo('#log');
});
</script>

</body>
</html>

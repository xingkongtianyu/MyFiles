:parent Selector

Categories: Selectors > Content Filter

parent selector

version added: 1.0jQuery(':parent')

描述： 选择所有含有子元素或者文本的父级元素。

这个真好和 :empty相反。

需要注意的一件重要的事情是:parent (和 :empty)的子元素包括文本节点。

W3C的建议<p>元素都至少有一个子节点，即使这个子节点是文字（见http://www.w3.org/TR/html401/struct/text.html＃edef-P）。另一方面，其他一些元素是空的（即没有子节点）：例如，<input>, <img>, <br>, and <hr>。

Example:

Finds all tds with children, including text.

<!DOCTYPE html>
<html>
<head>
  <style>
  td { width:40px; background:green; }
  </style>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  <table border="1">

  <tr><td>Value 1</td><td></td></tr>
  <tr><td>Value 2</td><td></td></tr>

</table>
<script>$("td:parent").fadeTo(1500, 0.3);</script>

</body>
</html>
